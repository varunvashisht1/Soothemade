# Prompt 05 — Quarterly review

## Use when

End of Q1 / Q2 / Q3 / Q4. ~13 weeks after launch. Strategic check-in: what's working, what's not, where to pivot.

## Time

10–15 min on Claude Pro

## Prompt (paste into Claude)

```
Help me do a quarterly review of my new-parent digital-products
Etsy + Gumroad shop. Be honest. I don't need to be coddled. If a
strategy isn't working, say so.

THIS QUARTER (Q[X] of [year]):

PRODUCTS LIVE: [list with launch dates]
PRODUCTS THIS QUARTER LAUNCHED: [list]

PERFORMANCE:
- Total Etsy sales: $[X]
- Total Gumroad sales: $[X]
- Best-selling product: [name + sales]
- Worst-selling product (0–1 sales): [name(s)]
- Total Pinterest impressions: [X]
- Total blog page-views: [X]
- Email subscribers: [X]
- Conversion rate (Etsy views → sales): [X%]
- Top 3 traffic sources to blog: [list]

NOTABLE FEEDBACK:
- Positive themes from reviews: [paste]
- Negative themes: [paste]
- Specific buyer requests for new products: [paste]

WHAT I'M WORRIED ABOUT:
[1–3 sentences on your top concern]

WHAT I'M FEELING GOOD ABOUT:
[1–3 sentences]

CONSTRAINTS:
- I have 20 min/week ops time
- I'm on Claude Pro ($20/mo), not Max
- I'm not paying for Etsy ads
- My Canva Pro expired [or still active]

Please give me:

1. **The honest read**: One paragraph. What's actually happening
   here. Be direct. Don't soften.

2. **Top 3 things working**: Specific products / channels / tactics
   that should get more attention.

3. **Top 3 things NOT working**: Specific products / channels /
   tactics that should be killed, deprioritized, or revised.

4. **Pivot recommendations** (if any): If the quarter's data
   suggests a strategic shift (e.g., "go all-in on sensory cards
   variants, drop the postpartum lane"), say so.

5. **Specific next-quarter actions**: 5 concrete things to do in
   the next 13 weeks. Each one specific (not "improve marketing"
   — "launch P01 Spanish-language variant by Wk 17").

6. **Pivot triggers**: What metric, if I don't see by next quarter,
   should I treat as a "kill the shop" signal? Don't be vague.

7. **One question to think about**: A strategic question this
   quarter raised that I should sit with.
```

## Expected output

A 600–1000 word strategic review. Direct, not sycophantic.

## After you run this

1. Save the output to `publishing/q[X]_review.md`
2. Update top-30 if a major pivot is suggested
3. Update `publishing/calendar.md` if scheduling needs to shift
4. Update `publishing/metrics.md` with the new pivot triggers
5. Pick the 5 specific actions and schedule them across the next quarter

## Pivot trigger defaults (use these unless overridden)

- **Q1**: <5 sales total = serious concern, refine
- **Q2**: <$200 revenue total = consider niche pivot or aesthetic refresh
- **Q3**: <$500 revenue total = pivot or wind down
- **Q4**: <$1000 revenue total = wind down or major strategic reset
