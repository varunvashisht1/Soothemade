## The niche (1 paragraph)

New-parent / baby-care digital products. Etsy + Gumroad split. Slow-drip publishing (~1 product/week). Aesthetic differentiation via warm-vintage / 70s-90s-coded palette to ride Pinterest 2026 trend (vs saturated minimalist-Goodnotes). Brand voice: "a slightly older friend who's been through it" — first-person, warm-direct, never preachy, no toxic positivity.

## The brand voice (paste this when asking Claude for any copy)

```
Brand voice: first-person ("I made this when..."), warm-direct, never
preachy. Specific concrete observations beat generalities. No "mama",
"mommy", "bump", "journey", "blessed", "enjoy every moment", "experts
say", "should/must/have to". Lead with a specific moment, not an
abstraction. One idea per paragraph. Soft sells, not urgency theater.
Disclaim once at the footer for YMYL-adjacent content.
```

## The visual identity (paste this when asking for design decisions)

```
Visual palette: warm vintage. Background warm ivory (#F5EFE6). Text
deep umber (#2E2A24). Accents terracotta (#B4654A), sage olive
(#8C9B7A), wheat (#D4A574). Typography: Fraunces for headings, Inter
```

## The top-tier opportunities (the products that should always be prioritized)

Pinterest 2026 trends to weight heavily in any new product decision:

- vintage baby clothes 90s: +600%
- daily routine chart for kids: +575%
- cognitive worksheets: +540%
- no phone summer: +340%
- slow motherhood: +310%
- positive discipline: +295%

When evaluating a new product idea, score against:
1. Demand (Pinterest trend × pain-point intensity)
2. Competition density on Etsy (lower is better)
3. Buildability with text + CSS + Canva (no paid image gen)
4. Margin (price ceiling × repeat-purchase potential)
5. YMYL safety (stay in organization/planning lane, NOT advice lane)

## Categories to AVOID

- Pure pregnancy planners (saturated)
- Generic newborn feeding trackers (saturated)
- Generic baby shower games (race-to-bottom)
- Sleep training methods (YMYL bear-trap)
- Feeding schedule prescriptions (YMYL)
- Baby name guides (saturated, low product-pull)
- Pure aesthetic nursery prints (need paid image gen)
- Photo-album memory books (need paid image gen)

## Underserved buyer segments (high-WTP, high-emotion)

- Working moms returning from leave (high income, time-poor)
- NICU / preemie parents (desperation buyers)
- Twin / multiples parents (small but loyal)
- IVF / high-risk pregnancies (high WTP)
- C-section recovery (specific gap)
- Pregnancy after loss / rainbow baby (sensitive, careful tone)
- Partner / non-birthing parent (almost no competition)

## Etsy SEO patterns that work (when writing new titles)

- 13 tags max, 20 chars each
- Title up to 140 chars
- Lead with category keyword (left-most matters most for ranking)
- 3–5 keyword synonyms in title (Etsy treats title + tags as combined keyword pool)
- Include format word ("Printable", "PDF Download", "Canva Editable", "Instant Download")
- Audience tag at end ("First Time Mom Gift", "Boho Baby")
- Aesthetic word drives style-matching browse traffic ("Minimalist", "Boho", "Botanical")

## The "always check" rule

Before launching any new product, ask Claude:

```
Quick sanity check: I want to launch [PRODUCT IDEA]. Based on the
knowledge base, is this:
1. In a saturated category I should avoid?
2. YMYL-adjacent? If so, can I keep it in the organization/planning
   lane?
3. Buildable with text + CSS + Canva (no paid images)?
4. Aligned to any current Pinterest rising trend?
5. Cross-sellable with existing products?
```

If 2+ red flags, don't launch.
