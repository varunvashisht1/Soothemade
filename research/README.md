| 1 | Reddit pain-phrase mining | `01-reddit-mining/` | Find exact phrases new parents use to describe unmet needs |
| 2 | Etsy top-seller teardown | `02-etsy-teardown/` | Reverse-engineer what's actually selling, find gaps |
| 3 | Blog SERP teardown | `03-blog-serp/` | Identify long-tail queries with rank-able gaps |
| 4 | Pinterest trends | `04-pinterest-trends/` | Search trend & seasonal pull |
| 5 | Customer journey map | `05-customer-journey/` | Pregnancy → toddler journey + emotional states + purchases |
| 6 | Problem → product matrix | `06-problem-product-matrix/` | Synthesis: scored opportunity list |

Final synthesis: `research/REPORT.md` — the Day 3 owner-review gate.
