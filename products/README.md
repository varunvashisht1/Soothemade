# Products

Built in Phase 2 (Days 4–7). One folder per product.

## Structure

```
products/
  P01_<slug>/
    product.pdf             ← deliverable
    mockup.png              ← Etsy thumbnail
    etsy-listing.md         ← title, 13 tags, description (ready to paste)
    gumroad-sales-page.md   ← long-form sales copy
    pinterest-pins/
      pin-1.png, pin-2.png, ..., pin-5.png
    source/
      cover.canva-link.txt
      interior.html
      interior.css
      mockup.canva-link.txt
    README.md               ← per-product publishing checklist
```

## Numbering

`P01..P30` — built in priority order from `research/06-problem-product-matrix/top-30.md`. P01–P05 are the Wave 1 launch products.
