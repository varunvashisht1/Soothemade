# P02 — The "Saying No" Script Pack

> Wave 1 launch product. Lowest complexity, zero-competition niche, fastest to ship.

## Product summary

A 30-script + 5-template printable PDF for new parents who need exact words to set boundaries with visitors, family, and unsolicited advice-givers.

- **Pages:** ~20-page PDF
- **Price (Etsy / Gumroad):** $7
- **Buyer:** late-pregnancy and fourth-trimester parents
- **Format:** PDF for printing or screen-reading; cards 4.25"×5.5" for cutting
