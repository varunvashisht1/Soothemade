# Pinterest Pin Specs — P02

5 pin variants. All 1000×1500 (Pinterest's preferred 2:3 ratio).

Build in Canva using the brand palette (warm-vintage primary — see `brand/visual-identity.md`):
- BG: `#F5EFE6` (warm ivory) or `#FBF6EE` (off-white kraft)
- Text: `#2E2A24` (deep umber-black)
- Accent: `#B4654A` (terracotta) or `#8C9B7A` (sage olive)

Typography:
- Headlines: **Caslon Antique** or **Cormorant Garamond**, ~64pt
- Subline: **Inter**, ~22pt
- Hand-letter accent: **Caveat**, ~28pt — one word only
