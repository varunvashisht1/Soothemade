# P43 — 1st Birthday Planning Kit

A 20-page planning kit for the first birthday. 6-week countdown, theme ideas (low-effort, brand-aligned), guest planning, food + cake decisions, photo shot list, memory keepers, day-of timeline. Built to make the birthday feel like a milestone without making the parent into an event planner.

## Why this exists

The first birthday on Pinterest looks like a wedding. It is not. It is a day. The kit gives the parent the structure to plan without the inflation, with brand-aligned simplicity and the option to scale up or down.

## YMYL care

- Non-YMYL.
- Food + cake page mentions the standard allergens parents check before introducing them to others, without prescribing.

## Scope

Approximately 20 pages:

- Cover, note, "you do not have to make this huge" page
- 6-week countdown (one page per week)
- Theme ideas (simple, brand-aligned)
- Guest list + invite tracker
- Food + cake page
- Smash-cake-or-not decision page
- Photo shot list (with non-cliché options)
- Memory keepers (letter-to-baby at 1, family quote-book pages)
- Day-of timeline
- A final note
